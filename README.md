# AI Assistant για Βελτίωση Αναζήτησης στην Online Βιβλιοθήκη της ΕΑΓΜΕ

## Σκοπός

Το παρόν repository/document set περιγράφει το scope, τα αναγνωρισμένα τεχνικά ρίσκα και το στήσιμο του πιλοτικού περιβάλλοντος για το έργο `TEAM2_03`.

Το έργο αφορά την ανάπτυξη AI Assistant που λειτουργεί συμπληρωματικά στο υφιστάμενο βιβλιοθηκονομικό σύστημα KOHA της ΕΑΓΜΕ. Ο στόχος είναι η βελτίωση της αναζήτησης και της σύνθεσης πληροφορίας από τεχνικές εκθέσεις, χωρίς αντικατάσταση της υπάρχουσας αναζήτησης.

## Κεντρική τεχνική θέση

Το έργο δεν πρέπει να αντιμετωπιστεί ως απλό “chatbot πάνω σε PDFs”.

Ο βασικός τεχνικός κίνδυνος είναι η αξιοπιστία της εξαγωγής κειμένου από ανομοιογενή, παλαιά και συχνά προβληματικά PDFs. Για τον λόγο αυτό, το πιλοτικό περιβάλλον πρέπει να περιλαμβάνει quality-aware extraction pipeline και να δημιουργεί embeddings μόνο από περιεχόμενο που έχει περάσει quality gates.

## Definition of Done πιλοτικής φάσης

Η πιλοτική φάση θεωρείται ολοκληρωμένη όταν:

- έχουν εισαχθεί περίπου 100 εκθέσεις με τα αντίστοιχα metadata,
- έχει εκτελεστεί extraction pipeline,
- κάθε σελίδα ή chunk έχει quality status,
- έχουν δημιουργηθεί embeddings μόνο από αξιόπιστο περιεχόμενο,
- λειτουργεί basic hybrid search,
- ο AI Assistant απαντά με πηγές και σελίδες,
- καταγράφονται queries, answers και user feedback,
- υπάρχει extraction/search evaluation report.
